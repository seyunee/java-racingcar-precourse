# 자동차 경주 게임

##기능 목록
 * n대의 자동차를 입력받는다. (,)로 구분한다
 * 사용자는 몇번 이동을 시도할지 입력한다.

 * 자동차는 전진/멈춤 을 할 수 있다
 * 자동차는 이름이 있다.
 * 이름은 5자 이하이다
 * 자동차는 전진할때 이름을 출력한다.
 * 0~9의 랜덤값에서 값이 4 이상이면 전진하고 미만이면 멈춘다

 * 이동 시도가 모두 끝나면 우승한 자동차의 이름을 알려준다.

## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.
